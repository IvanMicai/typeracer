# typeracer

## The stack

- [Node.js](https://nodejs.org/en/)
- [Express](https://expressjs.com/)
- [Mocha](https://mochajs.org/)
- [WebSocket](https://github.com/websockets/ws)
- [eslint](https://eslint.org/)

## Dependencies

- Node.js v8+ and npm v5+ installed

## Running
```
npm install
npm start
```

You should be able to access:
- http://localhost:8080

## Commands

Run Tests
```bash
$ npm test
```

Run Lint
```bash
$ npm run lint
```
